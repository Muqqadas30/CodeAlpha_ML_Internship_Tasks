# CodeAlpha Machine Learning Internship

This repository contains all the tasks completed during my Machine Learning 
Internship at **CodeAlpha**. The internship focused on building hands-on 
experience in machine learning algorithms, model development, data 
preprocessing, and model evaluation techniques.

## 🔍 Task Details

### Task 1 — Credit Scoring Model
Predicts an individual's creditworthiness using classification algorithms 
(Logistic Regression, Decision Tree, Random Forest, XGBoost) based on 
financial history such as income, debt, and payment records.

### Task 2 — Emotion Recognition from Speech
Recognizes human emotions (happy, sad, angry, etc.) from speech audio using 
MFCC feature extraction and deep learning models (CNN + LSTM) trained on 
the RAVDESS dataset.

### Task 3 — Handwritten Character Recognition
Identifies handwritten digits using a Convolutional Neural Network (CNN) 
trained on the MNIST dataset, extendable to full character/word recognition.

### Task 4 — Disease Prediction from Medical Data
Predicts the likelihood of disease (breast cancer) using classification 
algorithms (Logistic Regression, SVM, Random Forest, XGBoost) on structured 
medical data.
